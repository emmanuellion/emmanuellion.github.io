Ce package va vous permettre de générer vos propres barre de progression en toute simplicité !

Voici les fichiers dont vous aurez besoins avec un détails du code se situant à l'intérieur de chacun des fichiers

  • Cliquez ICI pour télécharger le fichier css
    :root{
     --h_bar: 35px;
     --h_border: 5px;
     --bar_size: calc(var(--h_bar) - (var(--h_border)*2));
    }

    div[data-progress]{
     width: 0%;
     height: var(--bar_size);
     transition: width 2s ease-in-out;
     text-align: right;
     padding-right: 10px;
     font-size: 16px;
     line-height: var(--bar_size);
     font-weight: bolder;
     color: white;
    }

    .bar-progress{
     border-radius: 20px;
     width: 250px;
     height: var(--h_bar);
     border: var(--h_border) solid black;
    }

    .crop-all{overflow: hidden;}
  • Cliquez ICI pour télécharger le fichier JS
    let tmp = document.getElementsByTagName("div");
    for(let i = 0; i < tmp.length; i++){
     let div = tmp.item(i);
     if(div.dataset.bg){
      div.style.backgroundColor = div.dataset.bg;
      if(div.classList.value.includes("bar-progress")){
       div.style.borderColor = div.dataset.bg;
      }
     }
     if(div.dataset.progress){
      div.style.width = div.dataset.progress+"%";
      if(div.dataset.progress>13)
       div.innerHTML = div.dataset.progress+"%";
      else
       div.innerHTML = div.dataset.progress;
     }
    }

Ainsi, il ne vous reste qu'à mettre ce code html en remplacant comme bon vous souhaite les valeurs dans les attributs "data-bg" et "data-progress"

< div class="card row column_center flex_center" data-bg="#7D49F4">
 < div class="bar-progress crop-all" data-bg="#3B2273">
  < div data-progress="100" data-bg="#0ff05e">< /div>
 < /div>
< /div>
